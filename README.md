# Monsters - Admin panel

Project is admin panel for 'monsters' game, allowing to manipulate data and game statistics

## 1. How to start

### Install dependencies

```shell
npm install / yarn
```

### Prepare environment

## 2. How to build

```shell
npm run build / yarn build
```

If you even encounter strange build behavior, tsconfig is set to create build with cache. Set option `incremental` in
tsConfig to false
